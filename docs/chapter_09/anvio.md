# anvio

组学数据分析和可视化平台

## 安装 anvio

```bash
$ sudo apt install libgsl-dev python3.5-dev prodigal sqlite3
$ mkvirtualenv --python=/usr/bin/python3.5 anvio
(anvio)$ pip install numpy scipy cpython anvio
(anvio)$
```
